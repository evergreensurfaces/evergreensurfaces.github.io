EVERGREEN SURFACE SOLUTIONS — BLOG FOLDER
=========================================

Put your blog post .html files in THIS folder (one file per post).

How to add a post:
1. Create a file here, e.g.  blog/spring-deck-staining-tips.html
   (Easiest: copy an existing service page and replace the content.)
2. Open ../blog.html in an editor.
3. Find the commented "ADD BLOG POSTS HERE" block and copy the card snippet
   into the <div class="grid grid-3"> ... </div>, pointing the href at your
   new file, e.g.  href="blog/spring-deck-staining-tips.html"
4. Give it a title and one-line summary. Done.

Tip: keep links relative. From blog.html, a post is  blog/your-post.html .
From inside a post file, link back to the site with  ../index.html , etc.
